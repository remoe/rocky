Detailed Description

TerrainState creates all the Vulkan state necessary to render the terrain.

TerrainTileDescriptors ROCKY_NAMESPACE::TerrainState::defaultTileDescriptors

Default state descriptors for a terrain tile. This holds the "default" (i.e. empty) textures and uniforms that will populate a descriptor set when no other textures are available. Terrain tiles copy and use this until new data becomes available.
